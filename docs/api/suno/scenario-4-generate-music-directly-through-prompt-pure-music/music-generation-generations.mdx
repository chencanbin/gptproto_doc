## Overview

Generate music interface, can generate directly or continue to write musicIf you are continuing to write a user-uploaded music, you need to add `-upload` after the mv field, such as `chirp-v3-5-upload`

## Authentication

This endpoint requires authentication using a Bearer token.

<ParamField header="Authorization" type="string" required default="sk-***********">
  Your API key in the format: `Bearer YOUR_API_KEY`
</ParamField>

## Request Body

<ParamField body="gpt_description_prompt" type="string"  default="an anthemic dancepop song about dancing all night long">
  Gpt Description Prompt parameter
</ParamField>

<ParamField body="mv" type="string"  default="chirp-v3-5">
  Mv parameter
</ParamField>

<ParamField body="prompt" type="string" required default="">
  Prompt parameter
</ParamField>

<ParamField body="make_instrumental" type="boolean"  default="true">
  Make Instrumental parameter
</ParamField>

## Request Example

<CodeGroup>

```bash cURL
curl -X POST "https://gptproto.com/suno/generate" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
  "gpt_description_prompt": "an anthemic dancepop song about dancing all night long",
  "mv": "chirp-v3-5",
  "prompt": "",
  "make_instrumental": true
}'
```

```python Python
import requests
import json

url = "https://gptproto.com/suno/generate"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}

data = {
  "gpt_description_prompt": "an anthemic dancepop song about dancing all night long",
  "mv": "chirp-v3-5",
  "prompt": "",
  "make_instrumental": true

response = requests.post(url, headers=headers, json=data)
result = response.json()
print(json.dumps(result, indent=2))
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io/ioutil"
    "net/http"
)

func main() {
    url := "https://gptproto.com/suno/generate"

    payload := []byte(`{
"gpt_description_prompt": "an anthemic dancepop song about dancing all night long",
"mv": "chirp-v3-5",
"prompt": "",
"make_instrumental": true
}`)

    req, _ := http.NewRequest("POST", url, bytes.NewBuffer(payload))
    req.Header.Set("Authorization", "Bearer YOUR_API_KEY")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, err := client.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    body, _ := ioutil.ReadAll(resp.Body)
    fmt.Println(string(body))
}
```
