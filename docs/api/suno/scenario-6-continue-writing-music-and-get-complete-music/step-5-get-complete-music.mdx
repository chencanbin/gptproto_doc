## Overview

After writing music, get the complete music of the original music and the merged music

## Authentication

This endpoint requires authentication using a Bearer token.

<ParamField header="Authorization" type="string" required default="sk-***********">
  Your API key in the format: `Bearer YOUR_API_KEY`
</ParamField>

## Request Body

<ParamField body="clip_id" type="string"  default="186653a1-ef95-4db5-8808-af10247267ce">
  Clip Id parameter
</ParamField>

<ParamField body="is_infill" type="boolean"  default="false">
  Is Infill parameter
</ParamField>

## Request Example

<CodeGroup>

```bash cURL
curl -X POST "https://gptproto.com/suno/generate/concat/v2" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
  "clip_id": "186653a1-ef95-4db5-8808-af10247267ce",
  "is_infill": false
}'
```

```python Python
import requests
import json

url = "https://gptproto.com/suno/generate/concat/v2"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}

data = {
  "clip_id": "186653a1-ef95-4db5-8808-af10247267ce",
  "is_infill": false
}

response = requests.post(url, headers=headers, json=data)
result = response.json()
print(json.dumps(result, indent=2))
```


```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io/ioutil"
    "net/http"
)

func main() {
    url := "https://gptproto.com/suno/generate/concat/v2"

    payload := []byte(`{
"clip_id": "186653a1-ef95-4db5-8808-af10247267ce",
"is_infill": false
}`)

    req, _ := http.NewRequest("POST", url, bytes.NewBuffer(payload))
    req.Header.Set("Authorization", "Bearer YOUR_API_KEY")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, err := client.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    body, _ := ioutil.ReadAll(resp.Body)
    fmt.Println(string(body))
}
```
