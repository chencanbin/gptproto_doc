## Overview

This endpoint provides chat mode functionality.

## Authentication

This endpoint requires authentication using a Bearer token.

<ParamField header="Authorization" type="string" required>
  Your API key in the format: `Bearer YOUR_API_KEY`
</ParamField>

## Request Body

<ParamField body="stream" type="boolean" >
  Whether to stream the response
</ParamField>

<ParamField body="model" type="string" required>
  The model to use for the request
</ParamField>

<ParamField body="messages" type="array" required>
  Array of message objects for the conversation
</ParamField>

## Request Example

<CodeGroup>

```bash cURL
curl -X POST "https://gptproto.com/v1/chat/completions" \
  -H "Authorization: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
  "stream": true,
  "model": "gemini-2.5-flash-image-preview",
  "messages": [
    {
      "content": "Draw a horse",
      "role": "user"
    }
  ]
}'
```

```python Python
import requests
import json

url = "https://gptproto.com/v1/chat/completions"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}

data = {
  "stream": true,
  "model": "gemini-2.5-flash-image-preview",
  "messages": [
    {
      "content": "Draw a horse",
      "role": "user"
    }
  ]
}

response = requests.post(url, headers=headers, json=data)
result = response.json()
print(json.dumps(result, indent=2))
```


```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io/ioutil"
    "net/http"
)

func main() {
    url := "https://gptproto.com/v1/chat/completions"

    payload := []byte(`{
"stream": true,
"model": "gemini-2.5-flash-image-preview",
"messages": [
{
"content": "Draw a horse",
"role": "user"
}
]
}`)

    req, _ := http.NewRequest("POST", url, bytes.NewBuffer(payload))
    req.Header.Set("Authorization", "Bearer YOUR_API_KEY")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, err := client.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    body, _ := ioutil.ReadAll(resp.Body)
    fmt.Println(string(body))
}
```
