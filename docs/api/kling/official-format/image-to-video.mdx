## Overview

This endpoint provides image to video functionality.

## Authentication

This endpoint requires authentication using a Bearer token.

<ParamField header="Authorization" type="string" required default="sk-***********">
  Your API key in the format: `Bearer YOUR_API_KEY`
</ParamField>

## Request Body

<ParamField body="prompt" type="string" required default="flow">
  Prompt parameter
</ParamField>

<ParamField body="negative_prompt" type="string"  default="">
  Negative Prompt parameter
</ParamField>

<ParamField body="image" type="string"  default="https://img.jpg">
  Image parameter
</ParamField>

<ParamField body="image_tail" type="string"  default="">
  Image Tail parameter
</ParamField>

<ParamField body="aspect_ratio" type="string"  default="1:1">
  Aspect Ratio parameter
</ParamField>

<ParamField body="mode" type="string"  default="std">
  Mode parameter
</ParamField>

<ParamField body="duration" type="string"  default="5">
  Duration parameter
</ParamField>

## Request Example

<CodeGroup>

```bash cURL
curl -X POST "https://gptproto.com/kling/v1/videos/image2video" \
```

```python Python
import requests
import json

url = "https://gptproto.com/kling/v1/videos/image2video"
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io/ioutil"
    "net/http"
)

func main() {
    url := "https://gptproto.com/kling/v1/videos/image2video"

    payload := []byte(`{
"prompt": "flow",
"negative_prompt": "",
"image": "https://img.jpg",
"image_tail": "",
"aspect_ratio": "1:1",
"mode": "std",
"duration": "5"
}`)

    req, _ := http.NewRequest("POST", url, bytes.NewBuffer(payload))
    req.Header.Set("Authorization", "Bearer YOUR_API_KEY")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, err := client.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    body, _ := ioutil.ReadAll(resp.Body)
    fmt.Println(string(body))
}
```
