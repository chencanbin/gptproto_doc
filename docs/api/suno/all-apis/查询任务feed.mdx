## Overview

This endpoint provides 查询任务(feed) functionality.

## Authentication

This endpoint requires authentication using a Bearer token.

<ParamField header="Authorization" type="string" required default="sk-***********">
  Your API key in the format: `Bearer YOUR_API_KEY`
</ParamField>

## Query Parameters

<ParamField query="ids" type="string">
  Task ID, multiple separated by comma `,`
</ParamField>

<ParamField query="remove_watermark" type="string">
  Whether to remove watermark
</ParamField>
