## Overview

[Image](https://platform.openai.com/docs/api-reference/images)Given a prompt and/or input image, the model will generate a new image.Related Guides: [Image Generation](https://platform.openai.com/docs/guides/images)Create an image based on the prompt.

## Authentication

This endpoint requires authentication using a Bearer token.

<ParamField header="Authorization" type="string" required default="sk-***********">
  Your API key in the format: `Bearer YOUR_API_KEY`
</ParamField>

## Request Body

<ParamField body="model" type="string" required default="gpt-image-1">
  The model to use for the request
</ParamField>

<ParamField body="prompt" type="string" required default="oxen and horses">
  Prompt parameter
</ParamField>

<ParamField body="n" type="integer"  default="1">
  Number of completions to generate
</ParamField>

<ParamField body="size" type="string"  default="1024x1024">
  Size parameter
</ParamField>

## Request Example

<CodeGroup>

```bash cURL
curl -X POST "https://gptproto.com/v1/images/generations" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
  "model": "gpt-image-1",
  "prompt": "oxen and horses",
  "n": 1,
  "size": "1024x1024"
}'
```

```python Python
import requests
import json

url = "https://gptproto.com/v1/images/generations"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}

data = {
  "model": "gpt-image-1",
  "prompt": "oxen and horses",
  "n": 1,
  "size": "1024x1024"
}

response = requests.post(url, headers=headers, json=data)
result = response.json()
print(json.dumps(result, indent=2))
```

```javascript JavaScript
const url = "https://gptproto.com/v1/images/generations";
const headers = {
  "Authorization": "Bearer YOUR_API_KEY",
  "Content-Type": "application/json"
};

const data = {
  "model": "gpt-image-1",
  "prompt": "oxen and horses",

fetch(url, {
  method: "POST",
  headers: headers,
  body: JSON.stringify(data)
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io/ioutil"
    "net/http"
)

func main() {
    url := "https://gptproto.com/v1/images/generations"

    payload := []byte(`{
"model": "gpt-image-1",
"prompt": "oxen and horses",
"n": 1,
"size": "1024x1024"
}`)

    req, _ := http.NewRequest("POST", url, bytes.NewBuffer(payload))
    req.Header.Set("Authorization", "Bearer YOUR_API_KEY")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, err := client.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    body, _ := ioutil.ReadAll(resp.Body)
    fmt.Println(string(body))
}
```
